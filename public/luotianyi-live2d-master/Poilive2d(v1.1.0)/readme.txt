== Description ==

添加一个洛天依Live2D人物在你的博客里
原模型来自手游 药水制作师
贴图修改：unsignedzhang
详细介绍： http://unsignedzhang.cn/2019/04/19/tianyi-live2d


== Installation ==

1. 上传 插件 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
